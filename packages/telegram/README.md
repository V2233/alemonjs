# [https://alemonjs.com/](https://alemonjs.com/)

跨平台开发的事件驱动机器人

## USE

- telegram

```sh
yarn add @alemonjs/telegram
```

- alemon.config.yaml

```sh
telegram:
  # 令牌 （必填）
  token: ''
  # 主人编号
  master_id: null
  base_api_url: null
  # 代理地址
  # proxy: 'http://127.0.0.1:7890'
  proxy: null
```
