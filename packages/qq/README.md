# [https://alemonjs.com/](https://alemonjs.com/)

跨平台开发的事件驱动机器人

## USE

- qq

```sh
yarn add @alemonjs/qq
```

- alemon.config.yaml

```sh
qq:
  # 账户
  qq: ''
  # 密码
  password: ''
  # 设备 1:安卓手机、 2:aPad 、 3:安卓手表、 4:MacOS 、 5:iPad 、 6:Tim
  device: ''
  # 签名
  sign_api_addr: ''
  # 版本
  ver: ''
  # 主人
  master_id:
    - ""
```
