# [https://alemonjs.com/](https://alemonjs.com/)

## USE

- qq-group-bot

```sh
yarn add @alemonjs/qq-group-bot
```

- alemon.config.yaml

```sh
qq-group-bot:
  # 编号
  app_id: ''
  # 令牌
  token: ''
  # 密钥
  secret: ''
  # 主人
  master_key:
    - ''
  # 事件订阅（有默认）
  intents: null
  # 分片（有默认）
  shard: null
```
